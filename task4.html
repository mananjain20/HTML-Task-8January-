What does auto do in grid rows?
In grid rows, auto means the row height adjusts automatically to fit the content inside it, growing or shrinking based on what is placed in that row.
Which row grows based on content?
The row with auto grows based on the content inside it.

Can rows have different heights?
Yes, each row can have a different height using any units like px, %, fr, auto.
What happens if content is very large?
If the content is very large, the auto row expands to fit all the content, making that row taller and pushing other rows down.
Where are grid rows useful in real layouts?
Grid rows are useful for headers and footers and can also be used to add horizontal sections on the page.
