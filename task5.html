Where does gap add space?
Gap adds space between grid items, both horizontally and vertically.
How is gap different from margin?
Gap controls spacing inside the grid layout, while margin adds space outside the elements.
Does gap affect container size?
No, gap does not affect the container size, it only distributes space between the items.
Can we use gap only for rows or columns?
Yes, rowgap and columngap can be used to control the gaps separately.
Why is gap preferred in Grid layouts?
Gap keeps spacing consistent and clean without extra margin specifications.
