Which layout breaks faster on small screens?
Layouts made with fixed units like px break faster on small screens, because they do not adjust to screen size, unlike flexible units like fr, %, or auto.
Which layout adjusts automatically?
The flexible layout that uses fr adjusts automatically according to the screen size.
Why is fixed layout risky?
A fixed layout is risky because it does not adapt to different screen sizes, so on small screens it can overflow, break alignment, and cause horizontal scrolling, making the design unusable on mobiles and tablets.
Where can fixed columns be useful?
Fixed columns could be used as sidebars or for adding advertisements to the side of web pages.
Which unit would you choose for a responsive website?
fr is the unit to use for a responsive website as it adjusts to the screen size.
